## ユーザーダイブ（脳内へ憑依）

### 検索した瞬間の脳内

> 「ダイエットしよう。まず1日何カロリー食べていいか知らないと始まらない」

- 行動の入口に立っている。まだ何も始めていない
- 「数字」を手に入れて安心したい
- 2-3回食事制限に挑戦して途中でやめた経験がある
- カロリー計算はしたことない。「基礎代謝」は聞いたことあるが自分の数字は知らない
- **今の感情: 「今度こそちゃんとやりたい」+ 「何が正しいかわからない」**
- **モード: 「計算機を探してる」。** 感情的なケアじゃなく、まず数字が欲しい

### ターゲット

**女性メイン。** 「摂取カロリー」検索層はダイエット関心層 → 大半が女性。CVも医療ダイエット（DMM）→ 女性ターゲット。カロリー・結果すべて女性ベースで設計。男性流入が多いとデータで判明したら改修。

### 競合サイトの体験との比較

| 比較軸 | 競合（フォーム型） | うち（LINE診断） |
|--------|------------------|----------------|
| 入力 | 性別・年齢・身長・体重・活動量の5項目 | 4問の対話型 |
| 結果 | 数字1つ（○○kcal）。出て終わり | カロリー + ものさしづくり + 理想の未来まで提示 |
| その先 | なし or アプリDL誘導 | GLP-1医療ダイエットへの橋渡し |
| 差別化 | LINE診断でカロリーを出す競合は不在 | 精度より体験価値で勝負 |

### カロリーの事実（ミフリン・セントジョー式 — 女性ベース）

30代女性(158cm, 55kg)の場合:

| 活動レベル | カロリー | 結果表示 |
|-----------|---------|---------|
| 座り仕事（×1.2） | 1,472 | 約 1,400 kcal |
| 軽い活動（×1.375） | 1,686 | 約 1,600 kcal |
| 中程度（×1.55） | 1,901 | 約 1,800 kcal |
| ハード（×1.725） | 2,116 | 約 2,100 kcal |

→ 身長・体重の個人差は±200kcal程度。結果には「目安」であることを注記。

## 設問設計

### Q1: 年代を教えてください

| 選択肢 | |
|--------|---|
| A | 20代 |
| B | 30代 |
| C | 40代 |
| D | 50代以上 |

**意図:** 計算機的な入り口。カロリー計算に年代を聞くのは当然→「なんでこれ聞くの？」が起きない。即答できる。フリクションゼロ。
**補足:** 年齢差は10歳で約24kcalなので結果分岐には使わない。「ちゃんと計算してくれてる感」を与える役割。
**結果分岐:** なし

### Q2: 普段の食事スタイルは？

| 選択肢 | |
|--------|---|
| A | 外食・テイクアウトが多め |
| B | 自炊が中心 |
| C | コンビニ・お弁当が中心 |
| D | 日によってバラバラ |

**意図:** 生活実態の確認。計算機っぽいデータ入力に見えるが、「コンビニ多め」を選ぶ行為自体が**自分の食生活を自覚する瞬間**になる。
**結果分岐:** なし

### Q3: ダイエット経験は？

| 選択肢 | |
|--------|---|
| A | 何度も挑戦した |
| B | 1-2回やったことある |
| C | 初めて |
| D | 今は情報集めてる段階 |

**意図:** 「何度も挑戦した」を選ぶ人が大半。**選択行為自体が「何度も失敗してる自分」の内省**になる。結果で「続かないのは意志のせいじゃない」と受け止める前フリ。
**結果分岐:** なし

### Q4（最終設問）: 日常の活動量に一番近いのは？

| 選択肢 | 結果タイプ | カロリー（女性ベース） |

**意図:** 結果分岐（唯一）。活動量の自然な4段階。女性ベースのカロリーを1つの数字でドンと出す。

## 結果カルーセル（4枚 × 4タイプ = 計16枚）

結果は4枚1組の画像カルーセル。各カードにヘッドライン + コピー + イラスト指示。
Card 2（不安喚起）とCard 4（欲求）がタイプ別。Card 1, 3は数字以外ほぼ共通。
体験者の声は結果には入れず、PRカルーセル（PR-4: 経験者の声）に任せる。
フッターの有無・形式はデザイナー判断。

## 弱点と対策

| 弱点 | 深刻度 | 対策 |
|------|--------|------|
| 女性ベースなので男性には数字がズレる | 低 | 男性流入が多いとデータで判明したら改修。初期は女性に集中 |
| カロリーのパーソナライズ限界（身長体重不問） | 中 | 「±200kcal程度の個人差」注記。「正確な数値は無料診療で」をCTA接続 |
| 情報探索段階→即CVは遠い | 高 | 友だち追加がファーストゴール。CVはナーチャリングで刈り取り |
| 体験談は実在のものが必要 | 中 | DMM側に実際の利用者データ・声を依頼。なければ「※イメージです」表記 |
