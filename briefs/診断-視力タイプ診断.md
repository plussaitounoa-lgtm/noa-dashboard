### ICLを気になり始めるきっかけ（4類型）

| 類型 | 具体的なきっかけ | 代表的な声 |
|---|---|---|
| **疲弊蓄積型** | コンタクトの管理・ドライアイ・角膜が傷つく | 「角膜がたわしで擦ったレベルで傷だらけと言われた」 |
| **ライフイベント型** | 育児・スポーツ・防災・マスク×メガネ | 「パッと目が覚めた時に赤ちゃんの可愛い様子がぼやけて見えないのが煩わしくて」「震災の時にメガネなくてもすぐ動けるようにしたい」 |
| **潜在→背中押し型** | 5年以上悩んで踏み切れなかった→信頼できる人・有名人がやっているのを見た | 「5年前から興味あったけど踏み切れなかった。合理的な人がやってるのを見て背中が押された」 |
| **強度近視の切実型** | 裸眼10cmしか見えない・眼鏡レンズが片眼10万超 | 「コンタクトは-10.5と-12.0。温泉でメガネなしでは何も楽しめない」 |

| ブロッカー | 代表的な声 |

| 手術への漠然とした恐怖 | 「ガチ手術だった。想定してたより怖かった」「手術直前に手が震えた」 |

| 年齢・タイミングへの迷い | 「5年後に貯めてやろうと思ってたが、遅すぎると分かった…貯めてたお金を似合うメガネ代にします」 |

## 1. ユーザーダイブ（脳内へ憑依）

### 検索した瞬間の脳内

> 「目が悪い。なんとかしたい。でも手術は怖いし高いし、まずできることだけ調べてみようかな」

- 視力回復トレーニングや「乱視 自力で治す」を調べている段階。ICLはまだ頭にない
- メガネ・コンタクトの生活に慣れてはいるが、「このままずっと？」という諦め感が薄くある
- コンタクトが面倒・目が乾く・毎朝の装着がストレス。でも「まあ仕方ない」で今日もつけてる
- 知人が「ICL受けた」と言っていた。気になってはいる。でも「自分の目に合うのか？」で止まってる
- **今の感情: 「情報だけ集めたい」＋「本当はすっきりしたい」**
- **モード: まだ動こうとしていない。でも気にはなっている**

### 想定される流入読者

ターゲットを絞って間口を狭めない設計（友だち追加増加が目的の初動のため）。ただし記事に入ってくる読者像として以下を想定する。

- コンタクト・メガネを日常的に使っていて、不便はあるが「まあ仕方ない」で今日もつけている
- 「視力回復トレーニング」「乱視 自力で治す」など自力解決を調べている段階。ICLはまだ視野に入っていない人も多い
- ICLを見聞きして気になってはいるが、「自分の目に合うのか？」で止まっている
- 強度近視（-5以上）ほど切実度が高く、行動意欲も高い傾向
- ※ 年齢層・性別分布はジャンル担当者に確認中。データが取れたら更新する

### ICLの事実（商材理解）

| ICLの機能 | ユーザーへの意味 |
|---|---|
| トレーニング不要・即日改善 | 「自力でなんとかしたい」層への現実的な答え |
| 強度近視対応（-18D） | 「自分には無理かも」の思い込みを解除 |
| 角膜を削らない・可逆性あり | 「手術が怖い」「永続的な変更が不安」の解消 |
| ドライアイ軽減 | コンタクトのストレスから解放される |
| 手術10〜15分・点眼麻酔のみ | 「大がかりな手術」のイメージを変える |
| 費用 約40〜80万円 | 「高い」→「コンタクト代と比較すると一生分の投資」で正当化される |

## 2. ものさしづくり

### 今のものさし（壊す対象）
「目が悪いのは仕方ない。眼鏡・コンタクトで矯正しながら一生付き合っていくもの。トレーニングや目薬でもしかしたら自力でなんとかなるかも」

※ 目が悪い → 眼鏡・コンタクトで補正する → ずっとこのまま（か自力でなんとかする）

### 新しいものさし（叩き込む）
「視力は治せる。角膜を削らず・取り出せる・10分で終わる。眼鏡・コンタクト以外の選択肢がある」

※ 近視は医療で治せる → 角膜を削らず・元に戻せる → 眼鏡なしの生活が選べる

### 新常識（ハンマー）
「近視は目の構造の問題なので、トレーニングや目薬では変わらない。視力は医療の力で治せる。」

- 体験者の声:「思ったより全然怖くなかった」

- 疑問はLINEで先に全部解決できます

## 9. 弱点と対策

| 弱点 | 深刻度 | 対策 |
|---|---|---|
| 準顕在層への訴求なのでCV(予約)まで時間がかかる | 高 | 友だち追加がゴール。CVはシナリオ配信で引き上げる設計にする |
| Type Aの結果「トレーニングは効かない」が否定的に映る可能性 | 中 | 「効かない」ではなく「効く人と効かない人がいる」の表現に留める。要観察 |
| 視力回復トレーニング記事（PV4,896）からの流入は自力志向が強い | 中 | Card3（なぜなのか）で「意志の問題ではない」と免責してから選択肢提示 |
| 「ICL」の言葉をPUで出していないためICLへの誘導に見えない | 低 | 意図的な設計。LINE登録後のシナリオでICLに誘導する |
| 対象記事のKWが視力一般系で、ユーザー温度感がバラつく | 低 | Q3の分岐で温度感を振り分け、タイプ別の結果カードで対応 |
