視力矯正に悩む準顕在層ユーザーをLINEに誘導し、診断を通じて先進会眼科への無料検査予約につなげます。

## 3. 設問パネル（5枚）

### Q1（視力レベル把握）
**設問:** あなたの裸眼視力はどのくらいですか？

| 選択肢 | テキスト |
|--------|--------|
| A | 裸眼だと数メートル先がほぼ見えない（0.05未満） |
| B | 手元は見えるが1〜2m先の文字が読めない（0.05〜0.1） |
| C | 遠くの看板・黒板が読めない（0.1〜0.3） |
| D | 少し悪い程度、日常はなんとかなる（0.3以上） |
| E | わからない / 測ったことがない |

### Q2（現在の矯正状況）
**設問:** 今、視力矯正をどうしていますか？

| 選択肢 | テキスト |
|--------|--------|
| A | コンタクトレンズを使っている |
| B | メガネを使っている |
| C | コンタクトとメガネを使い分けている |
| D | 何もしていない（裸眼） |

### Q3（悩みのペイン把握）
**設問:** 目のことで一番困っていることは？

| 選択肢 | テキスト |
|--------|--------|
| A | 視力がどんどん悪くなっている気がする |
| B | コンタクト・メガネが面倒・不便 |
| C | 目が疲れやすい・乾く |
| D | 左右で見え方が違う・ピントが合いにくい |

### Q4（理想の未来・感情接続）
**設問:** もし目の悩みが解決したら、一番変わってほしいことは？

| 選択肢 | テキスト |
|--------|--------|
| A | 朝起きたらすぐ見える生活がしたい |
| B | スポーツや旅行・温泉を、コンタクトやメガネなしで楽しみたい |
| C | ドライアイや目の疲れから解放されたい |
| D | 目のことを気にせず過ごしたい |

### Q5（結果タイプ分岐キー）
**設問:** 視力矯正に求めることで、一番近いのは？

| 選択肢 | テキスト | 分岐先 |
|--------|--------|--------|
| A | 今より見やすくなること | Q1視力レベルで眼鏡 or コンタクトに振り分け |
| B | 眼鏡の不便さと見た目をなくす | Q1視力レベルで眼鏡 or コンタクトに振り分け |
| C | 眼鏡・コンタクトから解放されること | TYPE C（ICL/レーシック） |

> **分岐ロジック補足（エンジニア向け）**  
> Q5=A or B → Q1がA/Bなら TYPE C寄り候補だがまずTYPE B（コンタクト）へ / Q1がC/D/Eなら TYPE A（眼鏡）へ  
> Q5=C → TYPE C（ICL/レーシック統合）へ直結
