**設問:** もし目の悩みが解決したら、一番変わってほしいことは？

| 選択肢 | テキスト |
|--------|--------|
| A | 朝起きたらすぐ見える生活 |
| B | メガネなしで運動や旅行をする |
| C | ドライアイや目の疲れから解放 |
| D | 目のことを気にせず過ごしたい |

### Q5（結果タイプ分岐キー）
**設問:** 視力矯正に求めることで、一番近いのは？

| 選択肢 | テキスト | 分岐先 |
|--------|--------|--------|
| A | 今より見えやすくなること | TYPE A（眼鏡） |
| B | 眼鏡の不便さと見た目をなくす | TYPE B（コンタクト） |
| C | 眼鏡やコンタクトから解放されること | TYPE C（ICL/レーシック） |

> **分岐ロジック補足（エンジニア向け）**  
> Q5=A or B → Q1がA/Bなら TYPE C寄り候補だがまずTYPE B（コンタクト）へ / Q1がC/D/Eなら TYPE A（眼鏡）へ  
> Q5=C → TYPE C（ICL/レーシック統合）へ直結
